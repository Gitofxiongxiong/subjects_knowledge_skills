# 当前知识库结构

线上根目录：

```text
/data/subject_knowbase/学科知识库
```

公开 skill 不应直接访问服务器文件系统；这些路径只用于理解目录结构。实际访问请使用 `/api/kb`。

`md≈` 表示该目录下递归包含的 OCR Markdown 页面数量，可近似理解为可读页数。

## 顶层目录

```text
学科知识库/
  content-generation_references/  md≈1        # 内部参考资料，通常不用
  脚本script/                     md≈0        # 内部脚本，通常不用
  高中化学/                       md≈13486
  高中数学/                       md≈16769
  高中物理/                       md≈14205
  高中生物/                       md≈9612
```

每个学科下都有 `_meta/`，包含目录索引等内部文件。不要直接读取 `_meta`；标题检索使用 `/search/catalog`。

## 使用建议

- 同步新课优先看 `新课讲义`、`新课同步讲义`、`教材`、`新课教案`。
- 一轮复习优先看 `一轮复习` 或 `一轮复习讲义`。
- 二轮专题优先看 `二轮复习` 或 `二轮复习讲义`。
- 考前冲刺优先看 `三轮复习`、`考前最后一课`、`考前20天冲刺攻略`。
- 需要题目和解析时，检索词中加入 `例题`、`典例`、`练习`、`解析`、`答案`、`专题`。
